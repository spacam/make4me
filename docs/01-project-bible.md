# Make4Me / Klíčník
## Project Bible

## 1. Vize projektu

Vytvořit nejlepší platformu pro poptávky a nabídky služeb v oblasti:

- výroby autoklíčů
- klasických klíčů
- autozámků
- zabezpečení vozidel
- alarmů
- imobilizérů
- nouzového otevírání vozidel

První verze bude určena pouze pro moje vlastní podnikání.

---

## 2. Hlavní cíl MVP

Zákazník zadá poptávku.

Poptávka přijde:

- do aplikace
- e-mailem
- push notifikací
- později SMS
